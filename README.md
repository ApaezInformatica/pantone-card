# Ejemplo de la famosa tarjeta pantone con Astro y tailwindcss

## Requisitos: Node 21.6.2

## Iniciar proyecto
* npm install o pnpm install
* npm run nodemon o pnpm run nodemon

* **src/fonts:** fuentes necesarias
